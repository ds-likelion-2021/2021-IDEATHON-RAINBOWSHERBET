# 아이디어소개
- 아이디어 설명 : 걷고싶길은 말 그대로 걷고 싶은 길. 시간을 내서라도 걷고 싶을 만큼 아름답고 다채로운 길들을 추천해주는 웹 서비스이다.
- 서비스 상세 소개 :
  1. 서비스 1
  원하는 길을 추천해주는 카테고리 검색 기능, 가보고 싶을 만한 여러 길들을 추천해주는 추천 길 기능
  2. 서비스 2
  나만 알고 있던 길들을 나누는 길 올리기 기능, 서로 의견을 주고 받으며 소통할 수 있는 커뮤니티 기능
